This is a generalized cellular automaton engine I made.

It is a sandbox for creating cellular automata on finite rectangular grids with finite states.

It's based off this previous version of the project which was worse in every way.

Here are some changes I want to make, sorted from highest priority to lowest priority

    • become:tile: Should select from a list of the icons

    • neighbor:seeking: Binary select for each of the icons
    • neighbor:matches: A rich number thing that can convert strings like ">1" to [2,3,4,5,6,7,8]. Also it needs to know neighborhood.length

    Notice how none of these are text inputs?
